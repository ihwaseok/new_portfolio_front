## 습관의 4단계 루프

```
신호 → 열망 → 반응 → 보상
```

| 단계 | 나쁜 습관 없애기 | 좋은 습관 만들기 |
|------|------------------|------------------|
| 신호 | 보이지 않게 하라 | 명확하게 하라 |
| 열망 | 매력 없게 하라 | 매력적으로 하라 |
| 반응 | 어렵게 하라 | 쉽게 하라 |
| 보상 | 불만족스럽게 하라 | 만족스럽게 하라 |

## 2분 규칙

> 새로운 습관은 2분 이내로 시작할 수 있어야 한다.

- "매일 운동하기" → "운동복 입기"
- "매일 독서하기" → "책 한 페이지 읽기"
- "매일 코딩하기" → "에디터 열기"

시작이 가장 어렵다. 시작을 쉽게 만들어라.

## 환경 설계

습관은 의지력보다 **환경**에 달려 있다.

- 책을 읽고 싶다 → 침대 옆에 책을 두어라
- 건강하게 먹고 싶다 → 과자를 눈에 안 보이는 곳에 두어라
- 코딩 공부하고 싶다 → 알림을 끄고 에디터를 바탕화면에 두어라

## 습관 쌓기 (Habit Stacking)

```
[현재 습관] 후에, [새 습관]을 한다.

예시:
- 아침 커피를 마신 후, 오늘의 목표 3가지를 적는다.
- 퇴근 후 자리에 앉으면, 오늘 배운 것을 기록한다.
- 양치 후, 감사 일기를 1줄 쓴다.
```

## 정체성 기반 습관

**"나는 ~하는 사람이다"** → 행동이 따라온다

- "운동을 해야지" ❌
- "나는 건강을 중시하는 사람이다" ✅
