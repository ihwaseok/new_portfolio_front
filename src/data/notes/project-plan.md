# 노트 퍼블리싱 사이트 계획

## 프로젝트 개요

Joplin 스타일의 웹 기반 노트 뷰어 개발

## 기술 스택

| 영역 | 기술 |
|------|------|
| Frontend | Vue 3 + TypeScript |
| 번들러 | Vite |
| 상태관리 | Pinia |
| 라우팅 | Vue Router |
| Markdown | marked |
| 스타일 | CSS Variables |

## 기능 목표

### MVP (1차)
- [x] 레이아웃 구성 (상단바 + 좌측 패널 2개 + 본문)
- [x] 노트북 트리 구조
- [x] 노트 목록 조회
- [x] 마크다운 렌더링
- [ ] 검색 기능

### 2차
- [ ] 코드 하이라이팅
- [ ] 목차(TOC) 자동 생성
- [ ] 프린트 최적화

## 일정

```
Week 1: 레이아웃 & 컴포넌트
Week 2: 데이터 연동 & 마크다운
Week 3: 검색 & 최적화

```
